# 🕸️ Listif India Business Scraper

A Python-based web scraper that extracts 2500+ business listings from the Listif India website across 250+ pages.

## 📊 Data Fields Collected

- Business Name
- Contact Number
- Email
- Website
- Contact Person
- Address
- Pincode

## ⚙️ Technologies Used

- Python
- BeautifulSoup
- Requests
